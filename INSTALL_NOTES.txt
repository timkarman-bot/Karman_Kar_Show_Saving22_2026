Karman Kar Shows & Events — v11 Sponsor Button Upcoming Fix

Purpose:
- Hide the "Sponsor This Show" button on the Upcoming page when that show has sponsorships turned off.
- Preserve the direct sponsorship URL protection that already shows sponsorship_closed.html.
- Preserve existing show/home/base behavior that already uses sponsorship_allowed(show).

Replacement files:
- templates/events.html

Install:
1. Copy templates/events.html into your project templates folder.
2. Run:
   python -m py_compile app.py database.py sponsorship_blueprint.py
   findstr /n /c:"sponsorship_allowed(upcoming_show)" templates\events.html
   findstr /n /c:"Sponsor This Show" templates\events.html
3. Commit:
   git add templates/events.html INSTALL_NOTES.txt
   git commit -m "Hide upcoming sponsor button when sponsorships disabled"
   git push origin main

Test:
- Sponsorships ON for upcoming show: Sponsor This Show should appear.
- Sponsorships OFF for upcoming show: Sponsor This Show should be hidden.
- Direct /sponsorship/<show-slug> should still show the sponsorship closed page when disabled.
