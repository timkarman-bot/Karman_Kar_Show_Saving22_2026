Karman Kar Shows & Events - Multi-Day Registration Slots Upgrade (v5)

WHAT THIS ADDS
- Multi-day / session-based registration capacity.
- Each show can have up to 5 registration slots/sessions from Manage Shows.
- Each slot has its own capacity, such as Saturday cap 30 and Sunday cap 30.
- Public registration requires a day/session selection when active slots exist.
- Full slots become unavailable automatically.
- Registration ZIP exports now include registration_slots.csv and slot columns on cars/intents.
- Keeps the prior cash show mode and export-any-show upgrade intact.

FILES INCLUDED
- app.py
- database.py
- templates/admin_shows.html
- templates/register.html
- templates/placeholder_claim.html
- templates/admin_show_mode.html

INSTALL SAFELY
1. Copy ONLY these files into the same paths in C:\karman-kar-shows.
2. Do not delete the full templates folder.
3. Overwrite the matching files only.
4. Run these checks:

   findstr /n "show_registration_slots" database.py
   findstr /n "registration_slot_id" app.py
   findstr /n "Registration Days / Sessions" templates\admin_shows.html
   findstr /n "Choose your day/session" templates\register.html

5. Then commit:

   git status --short
   git add app.py database.py templates\admin_shows.html templates\register.html templates\placeholder_claim.html templates\admin_show_mode.html INSTALL_NOTES.txt
   git commit -m "Add multi-day registration slots with per-day capacity"
   git push origin main

FATHER'S DAY WEEKEND SETUP
In Admin > Manage Shows, edit the church cruise-in show and add:

Slot 1:
- Label: Saturday Cruise-In
- Date: Saturday Father's Day Weekend
- Start: 4 PM
- End: 8 PM
- Cap: 30
- Active: checked

Slot 2:
- Label: Sunday Cruise-In
- Date: Sunday Father's Day Weekend
- Start: 7 AM
- End: 1 PM
- Cap: 30
- Active: checked

For this event, use no awards/voting. Set show type as full or popup as you prefer, but keep registration enabled.
