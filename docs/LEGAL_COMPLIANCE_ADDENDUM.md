# Karman Kar Shows & Events
# Data & Legal Compliance Addendum

---

# 📜 Data Collection Policy

We collect only necessary information:

Car Owners:
- Name
- Phone (required if opting into updates)
- Email (optional)

Attendees:
- First / Last name
- Phone (optional)
- Email (optional)
- Sponsor opt-in
- Updates opt-in

---

# 📬 Marketing Consent

Consent is stored with:

- consent_text
- consent_version

Sponsors may only contact attendees if:
sponsor_opt_in = 1

---

# 💳 Payment Compliance

Stripe handles:
- Payment processing
- PCI compliance
- Secure checkout

The system:
- Does not store credit card data
- Stores only Stripe session ID

---

# 🗂 Waiver Compliance

Paper-first waiver system.

Fields tracked:
- waiver_received
- waiver_received_at
- waiver_received_by

Provides timestamped legal record.

---

# 🔐 Data Protection Principles

- No selling of data
- Explicit opt-in required
- Audit trail preserved
- Snapshot exports for recovery
- Limited data collection

---

# 🛡 Audit Capability

Exports available:
- Votes CSV
- Snapshot ZIP (full database state)

Provides:
- Transparency
- Financial audit trail
- Legal defensibility
